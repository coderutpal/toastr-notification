Toastr Notification Setup Guide

This guide explains how to include and use Toastr.js notifications in your project using CDN links.

1. Include Toastr CSS in the <head>

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/toastr.js/latest/toastr.min.css" />

2. Include jQuery and Toastr JS before the closing </body> tag

<!-- jQuery (required for Toastr) -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Toastr JS -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/toastr.js/latest/toastr.min.js"></script>

3. Add Toastr Event Listener Script

This script listens for a showToastr event and shows the notification accordingly.

window.addEventListener('showToastr', function(event) {
    toastr.options = {
        closeButton: true,
        progressBar: true,
        positionClass: "toast-top-right",
    };

    if (event.detail.type === 'success') {
        toastr.success(event.detail.message);
    } else if (event.detail.type === 'error') {
        toastr.error(event.detail.message);
    } else if (event.detail.type === 'info') {
        toastr.info(event.detail.message);
    } else if (event.detail.type === 'warning') {
        toastr.warning(event.detail.message);
    }
});

4. Dispatching Notifications from Livewire (Laravel)

Use the following code inside your Livewire component to dispatch notifications:

$this->dispatch('showToastr', [
    'type' => 'success',  // success, error, info, warning
    'message' => 'Your message here'
]);

5. Dispatching Notifications in Plain JavaScript

Trigger a Toastr notification programmatically using vanilla JS:

window.dispatchEvent(new CustomEvent('showToastr', {
    detail: {
        type: 'success',   // success, error, info, warning
        message: 'Operation completed successfully'
    }
}));
